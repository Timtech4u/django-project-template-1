So you're starting a new Django project? You've come to the right place! This project template aims at making your start as quick and painless as possible so you can start writing your application instead of the boilerplate. Settings, basic templates, authentication, and authorization have all been implemented and tested so you can hit the ground running.

Things you'll get out of the box:

  • Django 1.11 is used to ensure you're using the latest and greatest djangoproject has to offer.
  • Python 3 is used with compliant code for a Python 2 project.
  • Basic authentication with user profiles and avatars and autorization including user groups and permissions are all implemented. Try registering in the top right corner to see.
  • Translation support is ready to go as all strings are translatable by default.
