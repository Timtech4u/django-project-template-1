So you're starting a new Django project? You've come to the right place! This project template aims at making your start as quick and painless as possible so you can start writing your application instead of the boilerplate. Settings, basic templates, authentication, and authorization have all been implemented and tested so you can hit the ground running.

  • Latest and Greatest Django Start with 1.11 to take advantage of all that Django has to offer.
  • Python 3 Out with the old and in with the future of Python!
  • Translation-Ready All text is translatable so your application can go international.
  • Authentication and Authorization From logging in to managing user permissions, this boilerplate gives you a strong start for a full-featured web UI. Use user groups, user profiles, and user avatars to personalize your application.
  • Frontend Build System Taking advantage of Webpack, you can compile your static assets for development and production builds as well as running live reload to watch for changes to your Javascript and CSS files.
  • Django Development Tools Tools such as django-debug-toolbar, django-extensions, and bpython are preinstalled to make tracking down bugs and performance issues much easier on you.
  • Fully Tested Pytest and coverage are configured with basic tests written to ensure when you start a new project it is 100% tested.
  • CSS With Tachyons Tachyons CSS is a next-level modular CSS framework allowing you to create explicit style from class names.
  • Deployment-Ready Deployment via Docker or directly to a server is one-step away with the config files and instructions ready to go.
